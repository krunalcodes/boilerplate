# Development Checklist

This checklist tracks the setup and configuration of essential features and tools for the boilerplate project.

## Code Quality & Developer Experience

- [ ] Setup ESLint, Prettier, TypeScript config, and EditorConfig
- [ ] Setup Husky and lint-staged for pre-commit hooks

## Backend Infrastructure

- [ ] Setup NestJS for the API
- [ ] Setup PostgreSQL and Drizzle ORM for the database
- [ ] Setup Better Auth with NestJS and Next.js
- [ ] Setup login via email and Google in NestJS
- [ ] Setup queuing using BullMQ in NestJS
- [ ] Setup stripe checkout

## Frontend

- [ ] Setup shadcn/ui with base UI components
- [ ] Create auth pages
- [ ] Create dashboard layout
- [ ] Create profile pages
- [ ] Create pricing and billing pages

## External Services

- [ ] Setup emails with Resend
- [ ] Setup Sentry for error logging
- [ ] Setup Stripe for payments

## DevOps & Deployment

- [ ] Setup Docker and Docker Compose for development/deployment
- [ ] Setup GitHub Actions for linting and deployment

## Documentation

- [ ] Update README with quick start guide
